## Key Interactions to Verify

- **Phase 0 prefs roundtrip**: open Settings → toggle theme → reload → theme persists. Then change accent color → reload → both persist.
- **Phase 0 scope-store on rapid drag**: drag the playhead from t=0 to t=last-act over 100ms; assert every scene boundary fires exactly one scope update (no missed, no doubled).
- **Wiki rework navigation**: open Wiki sidebar → click character → see Notes section nested → click `[[Aragorn]]` link in body → navigate to Aragorn's wiki page. Test `[[Char:Aragorn]]` type-prefix and `[[Unknown]]` fallback render.
- **Notes-as-sections enforcement**: attempt to POST `/api/entities` with `type='Note'` and no parentEntityId → expect 400. Existing orphans (if any) get reported by migration script, not auto-deleted.
- **World Map active-region**: drop a Location entity, draw a polygon region linked to it, scrub timeline so the location is active → region's stroke/fill updates to active style. Out of scope → dimmed style. Test this end-to-end with Playwright.
- **World Map hex_size lock**: PATCH `/api/maps/[id]` with `hex_size` change after a `map_hex_reveals` row exists → expect 422 + clear error message.
- **Cmd-K** (Phase 1.5): open palette via hotkey → type partial entity name → see ranked results → enter → navigate. Verify hotkey rebinding works.
- **Story Player** (Phase 1.5): hit play → playhead auto-advances at user-set speed → at least one subscriber window animates → pause works → manual scrub still works during playback.
- **T8b magic-link** (Phase 2): submit email → code logged to test-mode mailer → submit code → session cookie set → protected routes accessible.
- **T8b OAuth** (Phase 2): click Google login → mock provider returns claims → session created.
- **T8b user-A vs user-B isolation**: as user A, create entity X; as user B, GET /api/entities/X → 404 (not 200, not 403). Same for PATCH and DELETE attempts.

## Edge Cases

- **Preferences**: localStorage unavailable (private browsing) → app boots with defaults, no crash. Malformed JSON in localStorage → recover by rewriting defaults. SchemaVersion v1 stored, app at v2 → run migrations sequentially.
- **Scope store**: empty intervals list, single interval, t before first act, t after last act, t exactly on a scene boundary. Memoization invalidates on intervalsVersion bump.
- **Notes app**: 3-level nested folder, delete mid-level → cascade delete subtree, no orphan entries. Move entry from folder A to folder B → entry appears under B, gone from A. Empty folder rendering. Folder name with special chars.
- **Wiki rework**: `[[]]` empty link, `[[Name with [brackets]]]` nested, `[[Char:Aragorn]]` type prefix, `[[Aragorn|the king]]` future alias syntax (defer or define). Two characters with the same name → ambiguity resolution (fall back to type prefix or first-match warning).
- **World Map**: malformed polygon (self-intersecting), polygon with no Location, very large bitmap upload (size limit enforcement), B2 upload failure (graceful error), draw cancelled mid-action.
- **Hex fog (Phase 1.5)**: hex_size change attempt after first override → 422. Reveal driven by scope: scrub backward → already-revealed hexes stay revealed (revealed_at is monotonic). 5000 hexes perf benchmark.
- **T8b**: magic-link expired code (used or > 15 min) → reject. OAuth state-param mismatch (CSRF protection) → reject. Session cookie tampered → reject and redirect.

## Critical Paths (regression-grade, must work end-to-end)

- **REGRESSION 1 (wiki-rework)**: legacy `?focusEntityId=<note-uuid>` deep-link must resolve to a defined behavior under the Notes-as-sections model (lock behavior in spec, write E2E test).
- **REGRESSION 2 (T8b backfill)**: integration test that seeds the test DB with pre-T8b schema + rows, runs T8b migration, asserts every row has non-NULL `user_id` matching the first-user-uuid, and asserts the migration is idempotent (run twice = no-op).
- **CRITICAL (scope-store)**: Vitest unit test (drag math under fake timers) PLUS Playwright E2E test (real-browser drag at 100ms across all scene boundaries) — boil the lake per Issue 9 lock.
- **CRITICAL (notes folders)**: integration tests covering nested-folder CRUD, cascade delete, and move-between-folders per Issue 10 lock.
- **CRITICAL (T8b isolation)**: explicit per-endpoint specs for entities/intervals/relationships PLUS parametric sweep over the full endpoint registry as a backstop per Issue 11 lock.
- **HAPPY PATH (full demo)**: open app → import bitmap on World Map → draw 3 regions linked to 3 Locations → switch to Timeline → scrub through acts → see regions glow as locations enter scope → open Wiki sidebar → click character → see notes nested → click `[[Eowyn]]` hyperlink → navigate to Eowyn's page. (This is the demo flow; if any step breaks, the whole thing is unshippable.)
